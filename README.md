# AI Enhancer App 🚀

## Features
- Image Enhancement
- Video Enhancement
- History Tracking
- Download Results

## Tech Stack
- React (Frontend)
- FastAPI / Starlette (Backend)
- TensorFlow (AI Model)
- OpenCV + MoviePy (Video Processing)

## Run Locally

### Backend
cd backend
python app.py

### Frontend
cd frontend
npm start
